# Seller Catalog Web App

## Gettign Started

Install the dependencies:

```shell
yarn
```

Create an `.env` file:

```properties
VITE_API_URL=http://localhost:3001
```

Run the dev server:

```shell
yarn dev
```

Make your changes, test it.

Commit using:

```shell
yarn commit
```
